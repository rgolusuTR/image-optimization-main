# 🚀 Enable GitHub Pages - Simple Steps

## You're Currently Here:
✅ Signed in to GitHub
✅ On the Settings page
❌ Need to navigate to Pages section

## Follow These Exact Steps:

### Step 1: Click on "Pages" in the Left Sidebar
Look at the left sidebar in your Settings page. You'll see:
- General (you're here now)
- Collaborators
- Moderation options
- **Pages** ← Click this one!

### Step 2: Select GitHub Actions
Once you're on the Pages section:
1. Look for "Build and deployment" section
2. Under "Source", click the dropdown
3. Select **"GitHub Actions"**
4. The page will save automatically

### Step 3: Check Actions Tab
1. Click "Actions" at the top of the page
2. You'll see "Deploy to GitHub Pages" workflow running
3. Wait 2-3 minutes for it to complete

### Step 4: Access Your Site
Once deployment completes:
🌐 **https://rgolusutr.github.io/image-optimization-main/**
